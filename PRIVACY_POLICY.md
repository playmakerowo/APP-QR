# Privacy Policy

Last updated: May 8, 2026

## Overview

This application is designed exclusively for scanning QR codes using the device camera or images selected from the device gallery.

By using this application, you agree to the terms described in this Privacy Policy.

---

## Camera Permission

The application requests access to the device camera exclusively for QR code scanning functionality.

The camera is only used while the user actively scans a QR code.

No images, videos, or camera data are uploaded, transmitted, shared, or stored on external servers or cloud services.

---

## Gallery Permission

The application may request access to the device gallery or photo library to allow users to select images containing QR codes for scanning.

Selected images are processed locally on the device only.

No gallery content, photos, or scanned information are uploaded, shared, or stored externally.

---

## Local Storage

The application stores scan history locally on the user's device using local storage technologies.

This information remains exclusively on the user's device and is not transmitted to any remote server, cloud database, or third party service.

---

## Data Collection

This application does not collect personal information.

We do not:

* collect user accounts
* track user activity
* sell user data
* share information with third parties
* use cloud synchronization services
